# Personal Brand Website

A responsive multi-page static website built to showcase my freelance services as a web developer and data professional.

---

## 📸 Preview

> Homepage, About Page and Contact Page

---

## 📄 Pages

### Homepage (index.html)

The landing page of the website. It introduces who I am, what I do and provides navigation to other pages.

- Hero section with my name and title
- Services section showing what I offer (Web Development, Data Services, Document Formatting)
- Fully responsive navigation with hamburger menu on mobile
- Call to action buttons linking to About and Contact pages

### About Page (about.html)

A dedicated page that tells my story, showcases my skills and lists my work experience.

- Profile photo in a circular frame
- Bio paragraph describing my background
- Skill tags showing my technical and professional skills
- Experience timeline with my freelance history

### Contact Page (contact.html)

A page where visitors can reach out to me directly.

- Contact information (email, location, LinkedIn, GitHub)
- Contact form with JavaScript validation
- Glassmorphism card design with chocolate brown gradient background
- Error messages for empty fields and invalid email format
- Success message when form is submitted correctly

---

## ✨ Features

- Responsive design across mobile, tablet and desktop
- Mobile hamburger navigation that opens and closes smoothly
- CSS animations — page fade in, card hover lift, nav underline effect
- Smooth scrolling
- Hidden scrollbar
- JavaScript form validation
- CSS custom properties for consistent colors

---

## 🎨 Design Decisions

- **Color scheme:** Creamy white background (`#FAF8F2`) with dark navy (`#1a1a2e`) for navigation and footer, and chocolate brown gradient for the contact page
- **Typography:** Roboto and Roboto Mono from Google Fonts
- **Layout:** Mobile-first approach — styles written for mobile by default, then scaled up using `min-width` media queries

---

---

## 📐 CSS Concepts Demonstrated

- Flexbox for navigation and hero layout
- CSS Grid for services section
- `position: fixed` for sticky navigation
- `@media queries` for responsiveness
- CSS transitions and `@keyframes` animations
- `::after` pseudo-elements for nav underline effect
- `backdrop-filter` for glassmorphism effect
