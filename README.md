## 📌 Level 1 — Basic

### Task 1: Personal Brand Website

A multi-page static website built to showcase my freelance services as a web developer and data professional.

**Pages:**

- Homepage — Hero section, services overview
- About — Profile photo, skills, experience timeline
- Contact — Contact form with JavaScript validation

**Key Features:**

- Fully responsive across mobile, tablet and desktop
- Mobile hamburger navigation with smooth toggle
- CSS animations and hover transitions
- JavaScript form validation
- Smooth scrolling

**Technologies:** HTML5, CSS3, JavaScript, Font Awesome, Google Fonts

---

### Task 2: Developer Portfolio

A single-page developer portfolio inspired by a dark, bold design aesthetic. Built to showcase my projects and skills in a visually striking way.

**Sections:**

- Hero — Bold introduction with animated text
- Projects — Project cards with live and GitHub links
- Skills — Categorized skill cards
- About — Bio, profile photo and fun facts
- Contact — Contact form with validation

**Key Features:**

- Unique toggle switch mobile navigation with slide-in drawer
- Scroll reveal animations using Intersection Observer API
- Glassmorphism effects
- Form validation with JavaScript
- CSS custom properties (variables) for consistent theming
- Separate CSS files for variables, base styles, animations and media queries

**Technologies:** HTML5, CSS3, JavaScript, Fira Code Font, Font Awesome

## 📌 Level 2 — Intermediate

### Task 1: Blog Layout with Flexbox and Grid

A responsive blog layout that demonstrates the power of CSS Flexbox and CSS Grid working together.

**Sections:**

- Sticky navigation bar
- Hero header with search bar
- Featured blog post (Flexbox — side by side on desktop)
- Blog posts grid (CSS Grid — 2 columns on tablet, with sidebar on desktop)
- Sidebar with About widget, Categories and Popular Posts
- Footer

**Key Features:**

- CSS Grid for main layout (blog posts + sidebar)
- CSS Flexbox for navigation, featured post and footer
- Fully responsive — 1 column mobile, 2 columns tablet, 3 columns desktop
- Warm elegant color palette (Cremé, Bone, Oak, Clay)
- Playfair Display + Lato font combination
- Hover animations on cards

**Technologies:** HTML5, CSS3, Google Fonts (Playfair Display, Lato), Font Awesome

---

### Task 2: Todo Application

A fully functional Todo application that allows users to manage their daily tasks with data persistence using localStorage.

**Features:**

- Add new tasks by typing and clicking Add or pressing Enter
- Mark tasks as complete with a visual checkbox
- Delete individual tasks
- Filter tasks by All, Active or Completed
- Clear all completed tasks at once
- Task count updates automatically
- Data persists after page refresh using localStorage

**Key Concepts Demonstrated:**

- DOM manipulation
- Event listeners and event delegation
- Array methods (push, filter, find)
- localStorage (getItem, setItem, JSON.parse, JSON.stringify)
- Dynamic HTML generation from JavaScript

**Technologies:** HTML5, CSS3, JavaScript (Vanilla), localStorage API
